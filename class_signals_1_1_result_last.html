Detailed Description

template<typename ResultType>
class Signals::ResultLast< ResultType >

A class which gets a copy from the last result. We don't want a shared_ptr to result in somes cases, because emit() (which reset results) and results() are called in different threads. Thus would require an external lock to protect the result.
